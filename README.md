# 🧩 Maze Race Arena

Maze Race Arena is an interactive visualization project that compares multiple pathfinding algorithms in real-time. The project demonstrates how different algorithms explore and solve the same maze, highlighting their efficiency, behavior, and decision-making strategies.

---

## 🚀 Features

* 🔍 **Algorithm Race Visualization**

  * Breadth-First Search (BFS)
  * Depth-First Search (DFS)
  * A* Algorithm
  * Best-First Search

* 🎯 **Real-Time Execution**

  * All algorithms run simultaneously
  * First algorithm to reach the goal is घोषित as winner

* 🧠 **Heuristic-Based Pathfinding**

  * A* and Best-First use heuristic functions for optimized navigation

* 🎨 **Interactive UI**

  * Visual grid representation
  * Color-coded exploration paths
  * Smooth animations

* 🔄 **Dynamic Maze Generation**

  * Random maze generation
  * Guaranteed solvability

## 🧪 How It Works

* A grid-based maze is generated
* Each algorithm starts from the same point
* Algorithms explore the maze differently:

  * BFS → shortest path (level-wise)
  * DFS → deep exploration
  * A* → optimal path using heuristic
  * Best-First → greedy approach
* The first algorithm to reach the goal is declared the winner
